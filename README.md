**lorian-ct** 是一款面向学习场景的 AI 错题本工具。  
通过智能分析错题，帮助你高效归纳薄弱知识点，仅需几步即可在本地生成专属错题管理网站。

> 本项目由 **落雨纪元算法与数据研究中心团队** 开发，仅限学习用途。

---

## ✨ 特色

- 📦 **一键安装**：双击 `Setup.bat`，自动配置所有运行环境。
- 🚀 **一键启动**：双击 `start.bat`，立即在本地生成操作网站。
- 🧠 **AI 智能分析**：自动解析错题并提供参考建议（需自行甄别内容准确性）。
- 🔒 **完全本地运行**：数据不离开你的电脑，安全可控。

## 🚩 快速开始

### 环境要求

- Windows 10 及以上操作系统
- 稳定的网络连接（仅安装过程需要）

### 1. 下载与解压

下载 `lorian-ct.zip` 压缩包，**解压无需密码**，将文件夹放到你方便的位置即可。

### 2. 一键安装

首次使用前，双击运行文件夹内的 **`Setup.bat`**。  
脚本将自动下载并安装必需的依赖项，请保持网络通畅，等待窗口提示完成（或自动关闭）即可。

> 此步骤只需执行一次，后续可直接启动。

### 3. 启动服务

双击 **`start.bat`**，程序会自动启动并在默认浏览器中打开本地操作页面。  
若浏览器未自动弹出，可根据命令行窗口中显示的地址手动访问（通常为 `http://127.0.0.1:5000`）。

- 快捷入口：以后每次只需要点击 `start.bat` 即可立即使用。

### 4. 退出程序

在运行 `start.bat` 的命令行窗口中：

1. 按下键盘组合键 **`Ctrl + C`**
2. 出现提示后输入 **`y`** 并按回车
3. 程序安全关闭，本地网站随即停止

> 请避免直接关闭窗口，以防产生临时文件残留。

## ❓ 常见问题

**Q：启动后页面没有自动打开？**  
A：查看命令行窗口显示的本地地址（例如 `http://127.0.0.1:5000`），手动复制到浏览器访问。

**Q：`Setup.bat` 执行出错？**  
A：请确认网络连接正常，也可尝试关闭杀毒软件后重新运行。如问题持续，请访问官方博客或官网获取帮助。

**Q：AI 分析的结果一定正确吗？**  
A：不。AI 生成内容仅供学习参考，可能存在偏差或错误，务必自行判断真伪。
