**lorian-ct** 是一款面向学习场景的 AI 错题本工具。  
通过智能分析错题，帮助你高效归纳薄弱知识点。仅需双击 `start.bat` 即可在本地生成专属错题管理网站。

> 本项目由 **落雨纪元算法与数据研究中心团队** 开发，仅限学习用途。

---

## ✨ 特色

- 🚀 **一键启动**：双击 `start.bat`，自动在本地生成操作网站。
- 🌐 **网页配置 API**：可直接在网页中填入自己的 API Key，无需打开任何文件手动修改，大幅降低使用门槛。
- 🧠 **AI 智能分析**：自动解析错题并提供参考建议（需自行甄别内容准确性）。
- 🔒 **完全本地运行**：数据不离开你的电脑，安全可控。

## 🚩 从开源仓库构建

### 环境要求

- Windows 10 及以上操作系统
- 稳定的网络连接（仅安装过程及调用 AI 时需要）
- Python 3.11 及以上版本

### 1. 下载与解压

从 `https://github.com/bread-lorian/lorian-ct.git` 克隆项目。

### 2. 首次使用

双击运行项目内的 **`start.bat`**。  
脚本将自动下载并安装必需的依赖项，请保持网络通畅，等待服务启动即可。

> 此步骤只需执行一次，后续可直接启动。

### 3. 启动服务

双击 **`start.bat`**，程序会自动启动并在默认浏览器中打开本地操作页面。  
若浏览器未自动弹出，可根据命令行窗口中显示的地址手动访问（通常为 `http://127.0.0.1:xxxx`）。

> 🎉 **没错，就这么简单！** 以后每次只需要点击 `start.bat` 即可立即使用。

### 4. 配置你的 API（网页操作，无需改文件）

进入网页后，在设置页面中直接填入你自己的 API Key 并保存即可。  
**不再需要打开任何配置文件手动修改**，完全告别编程操作。

### 5. 退出程序

在运行 `start.bat` 的命令行窗口中：

1. 按下键盘组合键 **`Ctrl + C`**
2. 出现提示后输入 **`y`** 并按回车
3. 程序安全关闭，本地网站随即停止

> 请避免直接关闭窗口，以防产生临时文件残留。

## ❓ 常见问题

**Q：我想在Linux上使用此项目，我该怎么办？**  
A：只需要克隆本项目，并运行 `start.sh` 即可。请不要尝试发行版。

**Q：启动后页面没有自动打开？**  
A：查看命令行窗口显示的本地地址（例如 `http://127.0.0.1:5000`），手动复制到浏览器访问。

**Q：`Setup.bat` 执行出错？**  
A：请确认网络连接正常，也可尝试关闭杀毒软件后重新运行。如问题持续，请访问官方博客或官网获取帮助。

**Q：AI 分析的结果一定正确吗？**  
A：不。AI 生成内容仅供学习参考，可能存在偏差或错误，务必自行判断真伪。

**Q：我不懂编程，能用吗？**  
A：完全没问题！你只需要会双击 `start.bat`，然后在网页里填写 API Key 即可，其他一切交给程序自动处理。
